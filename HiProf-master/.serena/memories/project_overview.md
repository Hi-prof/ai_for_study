HiProf-master 是基于 RuoYi 3.9.0 改造的教务/教学管理系统。后端为多模块 Maven 工程，核心模块包括 hiprof-admin（Spring Boot 入口）、hiprof-system（系统管理）、hiprof-core（课程/成绩/会话等业务）、hiprof-tp（教案模块）、hiprof-portal 等。前端位于 hiprof-ui，使用 Vue 2 + Element UI + Vue CLI。数据库为 MySQL，默认配置在 hiprof-admin/src/main/resources/application-druid.yml。代码风格整体沿用 RuoYi：Java 使用驼峰命名、Controller/Service/Mapper 分层、MyBatis XML 管理 SQL；前端使用 Vue 2 Options API，接口层集中在 src/api，页面主要在 src/views。Windows 环境开发。
